# MMA Code Club

**Ideen bauen. Wirkung zeigen.**

Der Code-Club des Majors *Media Applications* an der FH Graubünden.
Für MMP-Studierende, die kreative Ideen in digitale Produkte übersetzen —
sichtbar, nutzbar und relevant für echte Zielgruppen.

[**mmacodeclub.ch →**](https://mmacodeclub.ch)

---

## 01 · Was hier entsteht

Webapps · KI-Workflows · Agenten · Datenstories · Live-Tools · Workshop-Software.

Projekte, die nach der Abgabe weiter funktionieren.

---

## 02 · Inhalte

| Bereich | Worum es geht |
|---|---|
| **Webapps** | Interfaces, Daten, APIs, Login, Karten, Dashboards, Prototypen. |
| **KI-Workflows** | Recherche, Struktur, Content, Code, Prototyping. |
| **Agenten & Automation** | Digitale Helfer, die recherchieren, sortieren, auswerten. |
| **UX & Workshops** | Menschen verstehen, Bedürfnisse sichtbar machen. |
| **Zukunft** | Portfolio, Selbständigkeit, Berufsbilder. |
